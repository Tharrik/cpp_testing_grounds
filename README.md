# C++ Testing Grounds
Project containing simple C++ examples to test. Uses C++17 standard.

- The project is divided in themed modules that contain simple functions that explains C++ concepts.
- The functions contain comments with the theory.
- These functions declare in their scope auxiliar classes used to implement examples of the concept.
- To further understand the concept, calling the function will show on runtime how the concept works.

## Usage
Simply go to the concept you want to understand and read the code. If you need it, you can call the function to see how it works at runtime.
